\section{Aufgetretene Probleme und offene Punkte}
\label{sec:appendix_c}
Während der Durchführung der verschiedenen Analysen und Experimenten konnten unterschiedliche Probleme festgestellt
werden:
\begin{itemize}
	\item Bei dem Versuch ein Bild in dem Dateiformat \textit{PNG} zu importieren, stürzte die für die Aufgabe vorgegebene
	externe Bibliothek \textit{pHash} mit unbekanntem Fehler ab. Die Ursache liegt in einer von \textit{pHash} verwendeten
	Bibliothek für das Laden von Bildmaterialien aus unterschiedlichen Dateiformaten \textit{CImg}.
	\item Bei dem dritten Datensatz führten Bildauflösungen größer als $2048xBel.$ zu nicht deterministischen Abstürzen der
	externen Bibliothek \textit{pHash}. Ursachen für diesen Fehler sind unbekannt.
	\begin{itemize}
		\item Daten aus dem \textit{Course media repository} (CMR) konnten aufgrund der hohen Auflösung nicht für die 
		experimentelle Untersuchung verwendet werden. Das Problem betrifft die folgenden Themen:
		\begin{itemize}
			\item Thema 1: Image Steganography
			\item Thema 7: Digital Camera Forensics
			\item Thema 9: Image Manipulation Detection
		\end{itemize}
	\end{itemize}
\end{itemize}
